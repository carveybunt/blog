## 组件命名

因为 setup中无法直接 name 所以在 setup 下面重新定义 name。
顺序不能乱，必须在 setup 下面。

```vue3
<script>
export default {
  name: 'componentName',
}
</script>

<script setup>
...
</script>

```

另外框架引用了 [vite-plugin-vue-setup-extend](https://github.com/vbenjs/vite-plugin-vue-setup-extend) 依赖，所以可以更方便的设置。

```vue
<script setup name="componentName">
...
</script>
```
