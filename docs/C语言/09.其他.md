## const
### 指针
```c
char s[]="s bu neng geng gai";
char str[]="Hello world";
const char* pS; // pS 指向的指针空间不能修改 pS不能被修改
char* const ptr = str; // ptr 指向不能改变，指向地址内容可以改变。 
pS = s;
// pS[0]="M"; // 不能修改指针地址内容，编译不通
Str[0]="M"; // 可以修改
// ptr = "ssssssssss"; // 不能修改指向，编译不通
ptr[0] = "N"; // 可以修改


```
