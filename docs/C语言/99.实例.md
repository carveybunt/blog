## 字符颠倒
将字符颠倒排序，包含有空格！
::: details
```c
char c[100],d[100]; // 未限制长度
gets(c);// 为获取输入字符串中的空格，得用 gets 或 fgets
int l;
l = (int)strlen(c);
printf("字符串长度为：%d\n", l);
for (int i = 0, j=l-1; i < l; ++i, --j) {
    d[j]=c[i];
}
puts(d);
return 0;
```
