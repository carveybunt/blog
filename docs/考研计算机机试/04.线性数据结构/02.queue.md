## 试题1

约瑟夫问题2
n个小孩围坐成一圈，并按顺时针编号为1,2,...,n，从编号为p的小孩顺时针依次报数，由1报到m，报到m时，这名小孩从圈中走出去；然后下一名小孩再从1报数，报到m时再出去。以此内推，直到所有小孩都从圈中走出去。请按出去的先后顺序输出小孩的编号。

输入：第一个是n，第二个是p,第三个是m（0<m，n<300）。
    最后一行是：0 0 0.

输出：按出圈的顺序输出编号，编号之间以逗号间隔。

样例输入：8 3 4
        0 0 0
样例输出：6,2,7,4,3,5,1,8
