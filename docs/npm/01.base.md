## nrm 源管理
替换阿里源
> npm config set registry=http://registry.npmmirror.com

安装 nrm

> npm install -g nrm
> nrm --version
> nrm ls // 查看源列表
> nrm use XXX

## yarn

> npm install --global yarn

### yarn 源管理
淘宝源
> yarn config set registry https://registry.npmmirror.com
yarn 源
> yarn config set registry  https://registry.yarnpkg.com
