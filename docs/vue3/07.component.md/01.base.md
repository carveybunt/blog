## 组件命名
因为 setup中无法直接 name 所以在 setup 下面重新定义 name。
顺序不能乱，必须在 setup 下面。
```vue3
<script setup lang="ts">
...
</script>

<script lang="ts">
export default {
  name: 'xxxxx'
}
</script>
```
