『职工』主要包括：
::: details
* 与企业订立劳动合同的所有人员，含全职、兼职和临时职工。
* 未与企业订立劳动合同，但由企业正式任命的企业治理层和管理层人员，如董事会成员、监事会成员等。
* 在企业的计划和控制下，虽未与企业订立劳动合同或未由其正式任命，但向企业所提供服务与职工所提供服务类似的人员。
:::
『职工薪酬』是指企业为获得职工提供的服务或解除劳动关系而给予的各种形式的报酬或补偿。包括
::: details
* 短期薪酬
* 离职后福利
* 辞退福利
* 其他长期职工福利
* 提供给职工配偶、子女、受赡养人、已故员工遗属以及其他受益人等的福利
<!-- more -->
:::
  ::: danger
  `工资薪金`可以全额扣除『企业所得税』。
  `社会保险费、公积金`可以全额扣除『企业所得税』。

     2008年1月1日起，`补充养老保险费`、`补充医疗保险费`分别不超过工资总额`5%`的部分可以扣除『企业所得税』。

  `职工福利费`不超过工资薪金总额的`14%`可以扣除『企业所得税』。
  `职工教育经费`不超过工资薪金总额的`8%`可以扣除『企业所得税』。
  `职工福利费`不超过工资薪金总额的`2%`可以扣除『企业所得税』。

  :::
## 职工薪酬的分类
### 短期薪酬
『短期薪酬』是指企业在职工提供相关服务的年度报告期间结束后12个月内需要全部予以支付的职工薪酬，因接触与职工的劳动关系给予的补偿除外。
* 「职工工资、奖金、津贴和补贴」，是按照构成工资综合的计时工资、计件工资、支付给职工的超额劳动报酬和增收节支的劳动报酬、为补偿职工特殊或额外的劳动消耗和引起特殊原因支付给职工的津贴，以及保证职工工资水平不受物价影响支付给职工的物价补贴等。
  > 其中企业按照短期奖金计划想职工发放的奖金属于短期薪酬，按照长期奖金计划向职工发放的奖金属于[其他长期职工福利](#其他长期职工福利)。
* 「职工福利费」是指企业向职工提供的生活困难补助、丧葬补助费、抚恤费、职工异地安家费、防暑降温等职工福利支出。
  ::: danger
  `工会经费`按工资总额`2%`计提缴纳。
  :::
* 「医疗保险费、工商保险费等社会保险费」是指企业按照国家规定的基准和比例计算，向社会保险经办机构缴纳的医疗保险费、工伤保险费。
* 「住房公积金」是指企业按照国家规定的基准和比例计算，向住房公积金管理机构缴存的住房公积金。
* 「工会经费和职工教育经费」是指企业为了改善职工文化生活、为职工学习先进技术及提高文化水平和业务素质，用于开展工会活动和职工教育及职业技能培训等相关支出。
* 「短期带薪缺勤」是指职工虽缺勤但企业任向其支付报酬的安排，包括：`年休假、病假、婚假、产假、丧假、探亲假`等。
  > 长期带薪缺勤属于[其他长期职工福利](#其他长期职工福利)。
* 「短期利润分享计划」是指因职工提供服务而职工达成的基于利润或其他经营成果提供薪酬的协议。
  > 长期利润分享计划属于[其他长期职工福利](#其他长期职工福利)。
* 其他短期薪酬，是指除上述薪酬以外的其他未获得职工提供的服务而给予的短期薪酬。
### 离职后福利
『离职后福利』是指企业为获得职工提供的服务而在职工退休或与企业接触劳动关系后，提供的各种形式的报酬和福利，[短期薪酬](#短期薪酬)和[辞退福利](#辞退福利)除外。
企业应当将离职后的福利计划分类为设定提存计划和设定收益计划。
* 「离职福后福利计划」是指企业与职工就离职后福利达成的协议，或者企业为向职工提供离职后福利制定的规章或办法等。
  * 「设定提存计划」是指向企业独立的基金缴存固定费用后，企业不在承担进一步支付义务的离职后福利计划。
  * 「设定收益计划」是指除设定提存计划以外的离职后福利计划。
### 辞退福利
『辞退福利』是指企业在职工劳动合同到期之前接触与职工的劳动关系，或者为鼓励职工自愿接受裁减而给予职工的补偿。
### 其他长期职工福利
『其他长期职工福利』是指除短期薪酬、离职后福利、辞退福利之外的所有职工薪酬，包括长期带薪缺勤、长期残疾福利、产期利润分享计划等。
## 科目设置
### 应付职工薪酬
一级科目：应付职工薪酬。
::: tip
贷方：登记已分配计入有关承办费用项目的职工薪酬,即「计提」。
贷方：登记实际发放的职工薪酬，包括扣还的款项等。
`贷增借减`，期末贷方余额，反应企业应付未付的职工薪酬。
:::
### 二级科目
工资、职工福利费、非货币性福利、社会保险费、住房公积金、工会经费、职工教育经费、带薪缺勤、利润分享计划、设定提存计划、设定收益计划、辞退福利等进行明细核算。

#### 国家规定计提标准的职工薪酬
##### 工会经费
根据《工会法》规定，每月全部职工工资总额的`2%`向工会拨缴经费,并在成本费用中列支，主要用于为职工服务和工会活动。
##### 职工教育经费
每月全部职工工资总额的`8%`计提，主要用于职工接受岗位培训。继续教育等方面的支出。
::: details
::: note
- 计提时
    
    计提时应借记：生产成本、制造费用、管理费用、销售费用、在建工程、研发支出等。
    贷记：应付职工薪酬————职工福利费 等科目。

    | &emsp;&emsp;销售费用                                                         |  1,740 |

##### 社会保险费
『社会保险费』包括`医疗保险费`、`养老保险费`、`失业保险费`、`工伤保险费`。企业承担的社会保险费，除`养老保险费和事业保险费`按规定确认为[离职后福利](#离职后福利)外，其他的社会保险作为企业的[短期薪酬](#短期薪酬)。
##### 住房公积金
『住房公积金分』为职工所在`单位为职工缴存`和`职工个人缴存`两部分，但其`全部属于职工个人所有`。

#### 短期带薪缺勤
- 累计带薪缺勤
  
  是指带薪权利可以结转下棋的带薪缺勤，本期尚未用完的带薪缺勤权利可以在未来期间使用。

  ::: details
  ::: note
  - 计提时
      
      计提时应借记：管理费用等科目。
      贷记：应付职工薪酬————带薪缺勤————短期带薪缺勤————累积带薪缺勤 等科目。

      | 科目                                                                     |  金额 |
      | :----------------------------------------------------------------------- | ----: |
      | 借：生产成本————基本生产成本                                             | 1,000 |
      | &emsp;&emsp;制造费用                                                     | 1,000 |
      | &emsp;&emsp;管理费用                                                     | 1,000 |
      | &emsp;&emsp;销售费用                                                     | 1,000 |
      | &emsp;&emsp;贷：应付职工薪酬————带薪缺勤————短期带薪缺勤————累积带薪缺勤 | 4,000 |

  :::
- 非累计带薪缺勤

  不能结转下棋的带薪缺勤，本期尚未用完的将予以取消，并且职工离开企业时也无权获得现金支付。
  > 职工休婚假、产假、丧假、探亲假、病假期间的工作通常属于非积累带薪缺勤。

  通常情况下，与非积累带薪缺勤相关的职工薪酬已经包括在企业每期向职工发放的工资等薪酬中，因此`不必额外作相应的财务处理`。
### 非货币性职工薪酬
* 企业以自产产品作为非货币性福利发放给职工的，根据收益对象，按照该产品的含税公允价值计入相关资产成本或当期损益。
  > 根据收益对象，可分为：生产成本、管理费用、销售费用、制造费用等。

  ::: details
  ::: note
  - 计提时

      将自产产品发放给员工
      | 科目                                         |    金额 |
      | :------------------------------------------- | ------: |
      | 借：生产成本                                 | 192,100 |
      | &emsp;&emsp;管理费用                         |  33,900 |
      | &emsp;&emsp;贷：应付职工薪酬————非货币性福利 | 226,000 |

      将非货币性福利结转主营收入(一般纳税人)
      | 科目                                                     |    金额 |
      | :------------------------------------------------------- | ------: |
      | 借：应付职工薪酬————非货币性福利                         | 226,000 |
      | &emsp;&ensp;贷：主营业务收入                             | 200,000 |
      | &emsp;&emsp;&emsp;&nbsp;应交税费————应交增值税(销项税额) |  26,000 |

      主营成本
      | 科目                       |    金额 |
      | :------------------------- | ------: |
      | 借：主营业务成本           | 180,000 |
      | &emsp;贷：库存商品————XXXX | 180,000 |

  :::
* 企业将拥有的房屋等资产无偿提供给职工使用的，根据收益对象，将该住房每期应计提的折旧计入相关资产成本或当期损益。
  ::: details
  ::: note
  - 例如，公司提供免费汽车给高级管理人员免费使用，提供租房免费用等

      汽车折旧：
      | 科目                             |   金额 |
      | :------------------------------- | -----: |
      | 借：应付职工薪酬————非货币性福利 | 20,000 |
      | &emsp;贷：累计折旧               | 20,000 |

      租房：
      | 科目                                   |   金额 |
      | :------------------------------------- | -----: |
      | 借：管理费用                           | 40,000 |

  :::
* 租赁住房等资产供职工无偿使用的，根据收益对象，将每期应付的租金计入相关资产成本或当期损益。
* 难以确认收益对象的直接计入当期损益和应付职工薪酬。
## 设定提存计划的核算
企业应当根据在资产负债表日向单独主体缴存的提存金，确认为应付职工薪酬，计入当期损益或相关资产成本。
贷记：`应付职工薪酬————设定提存计划`的下级科目。如基本养老保险费：

::: note

| &emsp;&emsp;贷：应付职工薪酬————设定提存计划————基本养老保险费 | 110,880 |
:::
