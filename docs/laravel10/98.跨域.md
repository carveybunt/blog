laravel 默认使用的是 fruitcake/laravel-cors 包来处理跨域。如果要访问受保护的路由，就需要确保在 cookie 中有 XSRF-TOKEN。然后浏览器在发送请求的时候，就会把 cookie 放在请求头中，发送给后端。

补充一点：在 config/cors.php 中可以设置受保护的路由。

`config/cors.php` :

```php
    'paths' => [
        'api/*',
        'sanctum/csrf-cookie'
    ],
```
