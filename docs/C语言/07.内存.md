::: danger
内存块因操作系统线数不同而不同，`32位系统` 4字节 对齐，即`8位`二进制；`64位系统`8字节 对齐，即`64位`二进制。
:::

市面上的CPU几乎都是小端存储，地位在前，高位在后。`两位一字节`。
::: note
如内存地址为：`0x012FFD00`，则在内存中表示为：`00 FD F2 01`。
:::
