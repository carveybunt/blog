## pnpm
[pnpm官网](https://pnpm.io/installation)

建议使用`pnpm 6`,因为它支持`node`12以上版本。
### 安装
* npm install -g pnpm
* brew install pnpm
* scoop install nodejs-lts pnpm
* npx pnpm add -g pnpm
  
> 以上任选其一进行全局安装

### 添加依赖
* pnpm add <pkg> 
  
  写入dependencies
* pnpm add -D <pkg>

  写入devDependencies
* pnpm add -g <pkg>

  全局
### 移除
* pnpm remove 包                            
 
  移除包
* pnpm remove 包 --global                   
  
  移除全局包

### 更新
* pnpm add -g pnpm

  自我更新

* pnpm up                
  
  更新所有依赖项

* pnpm upgrade 包        
  
  更新包
* pnpm upgrade 包 --global   
  
  更新全局包
