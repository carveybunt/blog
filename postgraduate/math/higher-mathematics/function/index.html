# 初等函数

将幂函数、指数函数、对数函数、三角函数以及反三角函数统称为基本初等函数。

由常熟和基本初等函数经过有限次数四则运算和有限次的函数复合步骤所构成并可用一个式子表示的函数，称为初等函数。
