# settings
## 自动换行
注意设置，千万别和其他的有冲突，不然不会生效！
```
{
    "workbench.sideBar.location": "right",
    "files.autoSave": "onFocusChange",
    "editor.formatOnSave": true,
    "[markdown]":{
        "editor.wordWrap": "on",
    },
    "diffEditor.wordWrap": "on",
    "editor.wordWrap": "on",
}
```
