## 传参

```php
Route::delete('/subject/{id}','delSubject')->whereUuid('id'); // 条件约束
```

## 特殊路由处理

### web

不存在的路由一律跳转到首页，而不是404. 必须放到路由的最后。

```php
Route::get('/{any?}', function () {
    return view('app');
})->where('any','.*')->name('index');
```
